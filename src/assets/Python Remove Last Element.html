    You can remove the last element from a Python array using several methods. Here are a few examples:

    Using pop()

    The pop() method removes and returns the last item in the list. If you don't need the removed element, you can simply call pop() without assigning it to a variable:

    python
    my_list = [1, 2, 3, 4, 5]
my_list.pop()  # Removes the last element (5 in this case)
print(my_list)  # Output: [1, 2, 3, 4]

    Using Slicing

    You can also use slicing to create a new list that excludes the last element:

    python
    my_list = [1, 2, 3, 4, 5]
my_list = my_list[:-1]  # Reassign the list excluding the last element
print(my_list)  # Output: [1, 2, 3, 4]

    Using del

    The del statement can also be used to remove the last element:

    python
    my_list = [1, 2, 3, 4, 5]
del my_list[-1]  # Removes the last element
print(my_list)  # Output: [1, 2, 3, 4]

    All these methods will modify the original list and remove the last element.

    my_list = [1, 2, 3, 4, 5]

    Using Negative Indexing:

    python
    my_list = [1, 2, 3, 4, 5]

    print(my_list[i])
